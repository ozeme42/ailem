# **App Name**: Ailem App

## Core Features:

- Family Overview: Display family members in gradient-background cards with achievement badges.
- Task Management: Implement a task management module with points, leaderboards, and advanced search/filtering.
- Family Calendar: Visualize the family calendar with colored event categories and upcoming event carousels.
- Reading Tracker: Track reading statistics and family profiles, allowing saving of favorite books and notes.

## Style Guidelines:

- Primary color: Blue (#3B82F6) to convey trust and stability, rooting from user prompt
- Background color: Very light blue (#F0F9FF).
- Accent color: Purple (#8B5CF6) to signal creativity, novelty and modern design trends.
- Body and headline font: 'Inter' for a clean and modern look (sans-serif).
- Employ relevant emojis for each section to improve clarity
- Subtle animations when users complete the tasks
